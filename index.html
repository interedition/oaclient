OAClient

This is an early demo of an application for annotating text resources on the web.

The approach follows the Linked Data paradigm and the model of the Open Annotation Collaboration. The application is implemented as a set of microservices, in accordance to the recommendations of Interedition.

Try it!

Please note that only plain text is supported for annotation targets so far.

Step 1

The client works as a bookmarklet. Please bookmark (drag into your bookmark bar) the following link: OAnnotate

Step 2

Visit a text file on the web, for example Project Gutenberg's Alice's Adventures in Wonderland.

Step 3

Load the client by visiting (clicking) the bookmark. Start annotating by selecting a span of text!
